# InSAR 页面简历面试准备 - 总览

## 项目背景

这是一个**地铁/隧道施工地表沉降监测系统**的前端页面。利用卫星 InSAR（合成孔径雷达干涉测量）技术，监测地面形变速度。工程人员通过这个页面判断哪些区域沉降严重、需要干预。

技术栈：React + TypeScript + Leaflet（地图） + ECharts（图表） + Tailwind CSS
