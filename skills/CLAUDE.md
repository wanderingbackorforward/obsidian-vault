# 学术论文写作技能系统

## 📋 项目概述

本项目旨在创建两个互补的Claude技能,用于辅助中文学术论文的撰写工作。通过将**排版规范**和**写作风格**分离为两个独立技能,实现更精确的控制和更灵活的使用。

### 核心理念
- **职责分离**: 排版归排版,风格归风格,各司其职
- **参考驱动**: 基于真实文档学习,而非抽象规则
- **实用优先**: 直接输出可用的.docx论文文件

---

## 🎯 两个技能的定位

### Skill 1: `thesis-formatting` (论文排版规范)

**核心职责**: 确保文档符合学术出版的格式要求

**学习对象**: 排版规范文档(.docx + .md)

**关注要素**:
- 页面设置: 页边距、纸张大小、方向
- 字体字号: 正文、标题各级、图表标题、页眉页脚
- 行距段距: 行间距、段前段后间距
- 页眉页脚: 内容、位置、字体
- 编号系统: 章节编号、图表编号、公式编号
- 特殊格式: 目录、摘要、参考文献的格式要求

**触发场景**:
- "按照学术规范创建论文"
- "帮我排版这个文档"
- "生成符合格式要求的论文模板"
- 用户明确提到"格式"、"排版"、"规范"等关键词

**输出**: 格式规范的.docx文件

---

### Skill 2: `thesis-writing-style` (论文写作风格)

**核心职责**: 生成符合学术规范的高质量中文内容

**学习对象**: 范文(.docx + .md)

**关注要素**:

1. **段落结构**
   - 段落长度和句子数量分布
   - 主题句位置(段首/段中/段尾)
   - 段落内部逻辑展开方式

2. **文本格式** (写作层面,非排版层面)
   - 段落缩进习惯(首行缩进2字符?)
   - 空行使用规则(章节间是否空行?)
   - 标点符号使用习惯(中文标点vs英文标点)

3. **语言风格**
   - 学术用语特点(客观、严谨、专业)
   - 句式结构(简单句vs复合句的比例)
   - 常用句式模板
   - 专业术语使用方式
   - 避免口语化表达

4. **论证逻辑**
   - 引言-正文-结论的组织方式
   - 章节过渡技巧
   - 论据呈现方式
   - 引用文献的表达习惯

**触发场景**:
- "写论文的引言部分"
- "用学术语言描述..."
- "按范文风格改写..."
- "生成文献综述"
- 用户需要撰写学术性文字

**输出**: 高质量的学术文本内容

## 📁 技能文件结构规范

每个技能必须放在自己的**子文件夹**中，不能直接放在 `skills/` 根目录。

正确结构示例：
```
skills/
└── thesis-formatting/       ← 技能名称子文件夹
    ├── SKILL.md             ← 技能主文件
    └── references/          ← 参考文件夹
        ├── 排版规范.docx
        └── 排版规范.md
```

错误示例（禁止）：
```
skills/
├── SKILL.md                 ← ❌ 不能放根目录
└── references/              ← ❌ 不能放根目录
```
