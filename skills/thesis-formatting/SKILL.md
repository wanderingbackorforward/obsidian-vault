# 学术论文排版技能 (Academic Paper Formatting Skill)

---
description: 按照学术专著排版规范创建和排版.docx论文文档。当用户提到"排版""格式""论文模板""学术规范""章节格式""图表编号"等关键词，或需要生成符合出版要求的.docx文件时触发。
---

## 概述

本技能用于按照专著/学术论文排版规范创建和排版 `.docx` 文档。**必须配合 `docx` 技能使用**，所有输出格式均为 `.docx`。

参考文件位于 `references/` 文件夹：
- `references/排版规范.docx` — 原始排版规范文档
- `references/排版规范.md` — 排版规范的文本版本

---

## 触发场景

以下任一情况应触发本技能：
- 用户要求"按照学术规范创建论文"
- 用户要求"帮我排版这个文档"
- 用户要求"生成符合格式的论文模板"
- 用户要求"按照模版格式写章节初稿"
- 用户提到"排版""格式""规范""模板""章节格式""图表编号"等关键词
- 用户需要输出 `.docx` 格式的学术文档

## 排版规范核心参数

### 标题层级

| 层级 | 字体 | 字号 | 加粗 | 对齐 | 行距 | 段前 | 段后 | 备注 |
|------|------|------|------|------|------|------|------|------|
| 章标题（第X章） | 黑体 | 三号 | **加粗** | 居中 | 单倍 | 24磅 | 18磅 | 章序号与章名间空一个字符 |
| 一级标题（X.X） | 黑体 | 小三号 | — | 顶左 | 单倍 | 24磅 | 6磅 | 序号与题名间空一个字符 |
| 二级标题（X.X.X） | 黑体 | 四号 | — | 顶左 | 单倍 | 12磅 | 6磅 | 序号与题名间空一个字符 |

### 正文
- Word 样式名：`论文正文`

### 图

| 属性 | 规范 |
|------|------|
| 图宽 | 12–14 cm（画图板中字约 20–28） |
| 图中文字 | 9磅（小五号） |
| 图题位置 | 图的**下方** |
| 图题字体 | 宋体 |
| 图题字号 | 五号 |
| 图题对齐 | 居中 |
| 图题行距 | 单倍行距 |
| 图题段前 | 6磅 |
| 图题段后 | 12磅 |
| 图序格式 | 图序与图名之间空一个字符宽度 |
| 示例 | `图 1.1 非线性构形状态转移过程示意图` |
| Word 样式名（图） | `论文图` |
| Word 样式名（图号） | `论文图号` |

### 表

| 属性 | 规范 |
|------|------|
| 表题位置 | 表格**上方** |
| 示例 | `表1-1土体参数` |
| Word 样式名（表名） | `表格头` |
| Word 样式名（表文字） | `表格文字` |

### 公式
- 公式**居中**排列
- 编号**右对齐**，格式：`（章号.序号）`，如 `（1.1）`
- 插入方式：在公式行末按 `Ctrl+Alt+Enter` 插入样式分隔符，使编号与公式同行显示
- 交叉引用快捷键：`Alt+S`

### 交叉引用方法

编号和引用的完整操作流程：

1. **章节编号**：从模版中拷贝章和节到目标文档，其他章节号通过格式刷形成固定格式
2. **正文及图表名**：通过格式刷或选取对应内容，点击样式面板中对应的样式名称
3. **图表及公式编号**：
   - 选择"插入" → "题注" → "新建标签"，将标签名改为"图"/"表"/"公式"
   - 设置编号格式，选择"包含章节号"，设置起始样式及分隔符
   - 选择"插入" → "交叉引用"进行引用
   - 更新域：选中域 → 右键 → "更新域"

### Word 样式名称速查表

| 内容类型 | Word 样式名 |
|----------|-------------|
| 正文 | `论文正文` |
| 图 | `论文图` |
| 图号 | `论文图号` |
| 表名 | `表格头` |
| 表文字 | `表格文字` |

## 使用方法

### 调用 docx 技能

本技能**必须配合 `docx` 技能**生成 `.docx` 文件。调用模式：

1. 先读取 `references/排版规范.md` 确认完整规范
2. 使用 `/docx` 技能创建或编辑 `.docx` 文件
3. 在调用 docx 技能时，**必须明确传递以下参数**：
   - 每个元素的字体、字号、行距、段前段后
   - 中英文字体分别设置（中文宋体/黑体，英文 Times New Roman）
   - 使用上述 Word 样式名称

### 示例调用

```
使用 docx 技能创建一个符合排版规范的论文章节，包含：

【章标题】黑体，三号，加粗，居中，单倍行距，段前24磅，段后18磅
【一级标题】黑体，小三号，顶左，单倍行距，段前24磅，段后6磅
【二级标题】黑体，四号，顶左，单倍行距，段前12磅，段后6磅
【正文】使用"论文正文"样式
【图题】宋体，五号，居中，单倍行距，段前6磅，段后12磅，样式名"论文图号"
【表题】使用"表格头"样式，置于表格上方
```

---

## 注意事项

1. **所有排版参数必须严格遵循参考文件**，不得自行调整或脑补
2. **输出格式必须为 `.docx`**，不接受其他格式
3. **中英文字体分别设置**：中文（宋体/黑体）和英文（Times New Roman）需独立配置
4. **必须配合 `docx` 技能使用**，不要尝试用其他方式生成文档
5. **章标题必须加粗**，一级和二级标题不加粗
6. **原始规范未明确规定的参数，不得自行添加**
